# Actividad 2: Práctica de maquetación

Clon educativo de la página principal de Google para la materia **Desarrollo Front-End**.

## Tecnologías

- HTML5 semántico
- CSS3 responsive
- JavaScript

## Funcionalidades

- Barra de búsqueda con sombra al pasar el cursor o recibir el foco.
- Búsqueda al presionar el botón o la tecla `Enter`.
- Botón para limpiar el texto.
- Botón «Voy a tener suerte».
- Diseño adaptable a computadoras y teléfonos.
- Estados `hover` y `focus` en botones y enlaces.

## Ejecución

1. Descarga o clona este repositorio.
2. Abre `index.html` en tu navegador.
